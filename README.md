# LoyaltyGuard AI: Automated RFM Segmentation System

This project implements a loyalty analysis model using the **RFM** (Recency, Frequency, Monetary) methodology to automatically segment a customer base. The primary goal is to identify key behaviors—such as VIP customers and users at risk of churn—to optimize retention strategies.

## 🚀 Key Features
* **Data Processing**: Cleaning and transformation of transaction logs.
* **RFM Segmentation**: Calculation of scores for each customer based on their last purchase, frequency, and total spend.
* **Categorization**: Automatic classification into segments (Champions, Loyal, At Risk, etc.).
* **Visualization**: Clear charts to support data-driven decision-making.

## 🛠️ Tools Used
* **Python 3.12**
* **Pandas**: For data manipulation and analysis.
* **Matplotlib / Seaborn**: For generating visual insights.
* **Jupyter Notebook**: The development environment for the analysis.

## 📂 Repository Structure
* `analisis_lealtad.ipynb`: Main notebook containing the full analysis and code.
* `requirements.txt`: List of necessary libraries to run the project.
* `Alerta_Clientes_Riesgo.xlsx`: Generated report of customers detected as "at risk".

## 📊 Results
This model allows businesses to immediately visualize which percentage of their portfolio requires urgent attention, enabling more efficient marketing budget allocation.
